CONTROL DE CORTE DE CABLE - V3.2

Suba todos estos archivos sueltos a la raíz del repositorio de GitHub Pages.

Cambios principales:
- La app exige iniciar sesión para registrar cortes.
- El responsable del corte se toma automáticamente del usuario activo.
- El auxiliar solo diligencia el responsable de chequeo y los datos del corte.
- Gerencia puede habilitar usuarios desde la app.
- Los roles se controlan por Firestore: OPERARIO, LIDER_LOGISTICO, GERENCIA.

Después de subir el ZIP descomprimido, copie el contenido de firestore.rules en Firebase > Firestore Database > Reglas y publique.
